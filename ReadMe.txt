 Steps to run back end

1. Install python 3.8 in your computer
2. Install following python modules using following command
  pip install<module_name>
  a. flask_cors
  b.werkzeug.security 
  c. flask
  d.flask_sqlalchemy 
  e. flask_migrate 
  f. flask_restful   
    g. flask_jwt 
  h. flask_apispec
  i. apispec

3. Go to the service folder in command prompt
  Type "python basicmodel.py"

Steps to run Front end

1.  Install latest version of node in your computer
2. type the command in command prompt "npm install -g @ionic/cli"
3. Go to the web folder in project. 
4. in the command prompt of web folder type the following command
type "npm install"
type "ionic serve"

End of instructions
